# Impact Report (Portfolio Sample)

This is a redacted, anonymized sample of an impact report I designed, wrote, and built end to end. It is part of a larger portfolio demonstrating my abilities across data storytelling, persuasive writing, and frontend data visualization.

---

## What This Sample Demonstrates

### Writing & Narrative Strategy
Impact reports live or die by their ability to translate numbers into meaning. This sample showcases my ability to craft clear, compelling narratives for funder and stakeholder audiences by structuring data-driven stories that are both credible and persuasive. 

### Data Visualization & Frontend Development
The interactive visualizations throughout this report were built from scratch using D3.js, CountUp.js, and JavaScript. Key features include:

- **Animated alumni growth graph** — a data-terminal-inspired visualization depicting cumulative network growth from 2003 to 2026, with a zooming camera effect and glowing dot trail
- **Multiplier effect network graph** — a force-directed D3 network animation showing how one organization's reach expands across Fellows, Alumni, and their broader connections across 100+ nodes
- **Animated stat counters** — CountUp.js-powered metrics that activate on scroll, making key numbers land with impact

### Design System & Frontend Engineering
The report is built on a custom design system with a defined color token library, typographic scale, and component architecture — all hand-coded in HTML, CSS, and JavaScript and hosted via GitHub Pages.
